  • 本章展示 IntelliJ IDEA 高度集成化的效果，降低 IntelliJ IDEA 入门时间

开发环境

  • JDK 7（理论上支持 JDK 6、JDK 7、JDK 8）
  • Mysql 5.6
  • Maven 3.1.1
  • Tomcat 7
  • Git 2.7.0.2-64-bit
  • IntelliJ IDEA 15.0.4
  • 所有编码：UTF-8
